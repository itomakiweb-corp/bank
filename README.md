## Flow

- git checkout master
- git pull origin master
- git branches, new branch
- 新しいブランチ上で必要な修正を実施
- emulatorなどで動作確認
- git commit
- git push
- web pull request, reviewersを指定
- web reviewersのレビューとマージ待ち
- 最初に戻る
